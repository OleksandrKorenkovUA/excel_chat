# Практичний курс роботи з даними

## Аналіз структури файлу test2020.xlsx

**Загальна інформація:**
- Кількість записів: 100 товарів
- Кількість колонок: 10

**Структура колонок:**

| Колонка | Тип | Опис |
|---------|-----|------|
| ID | float64 | Унікальний ідентифікатор товару (1001-1099) |
| Категорія | object | Група товару (Ноутбук, Відеокарта, Планшет тощо) |
| Бренд | object | Виробник товару (Apple, ASUS, Logitech тощо) |
| Модель | object | Назва конкретної моделі |
| Опис | object | Опис призначення товару |
| Колір | object | Колір товару |
| Специфікація | object | Технічні характеристики |
| Ціна_UAH | float64 | Ціна в гривнях (250-89,999 UAH) |
| Кількість | int64 | Кількість одиниць на складі |
| Статус | object | Наявність (В наявності, Закінчується, Під замовлення, Резерв) |

**Ключові категорії:**
- Ноутбуки (5 позицій)
- Відеокарти (5 позицій)
- Процесори (5 позицій)
- Монітори (5 позицій)
- Планшети (5 позицій)
- Роутери (5 позицій)
- Принтери (3 позиції)
- Клавіатури (5 позицій)
- Миші (5 позицій)
- Гарнітури (5 позицій)
- Пам'ять (RAM/SSD - 10 позицій)
- Корпуси/Охолодження (9 позицій)
- Кабелі/Хаби (5 позицій)
- Аксесуари/Software (5 позицій)

**Типові значення:**
- Середня ціна: 13,120 UAH
- Ціни: від 250 UAH (термопаста) до 89,999 UAH (MacBook Pro 14)
- Кількість: від 1 до 867 одиниць
- Статуси: В наявності (91), Під замовлення (5), Резерв (2), Закінчується (1)

## Корисні поради для аналізу

1. **Для пошуку:** Використовуй фільтрацію по колонках "Категорія", "Бренд", "Статус"
2. **Для обчислень:** Використовуй агрегації по "Ціна_UAH" (sum, mean, min, max)
3. **Для аналізу наявності:** Фільтруй по "Кількість" для знаходження товарів, що закінчуються
4. **Для аналізу виручки:** Створюй розрахункову колонку "Ціна_UAH × Кількість"
5. **Для порівнянь:** Групуй по "Категорія" або "Бренд" для порівняльного аналізу
